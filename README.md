# Al-Quds: Witness & Memory

This is a web project that serves as a digital memorial and interactive experience about Al-Aqsa and its defenders.  
It started from an open-source HTML/CSS/JS template and has been heavily modified to fit the theme of history, memory, and dignity.

## Features

- **Black opening screen with text**: invites reflection before exploring content
- **Watery cursor reveal effect**: images and backgrounds appear fluidly as you move your mouse
- **Sections with images and text**: telling the story of Al-Aqsa and its defenders
- **Responsive design**: works on desktop and mobile
- **Accessible fonts**: Noto Serif Arabic and Crimson Pro used for readability
